# vue-apng

a vue component for showing animating png.

# install

```bash
npm i vue-apng
```

```js
import VueApng from 'vue-apng'

Vue.use(VueApng)
```

or

```
import VueApng from 'vue-apng'

new Vue({
  components: {
    VueApng,
  }
})
```

```html
<vue-apng class="apng" src="imgUrl"></vue-apng>
```
